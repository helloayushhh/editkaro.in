# editkaro.in

a modern portfolio website for a creative video editing agency.

designed to showcase video editing work through a clean interface, smooth animations, and an interactive portfolio experience.

## features

- cinematic hero section
- interactive portfolio filters
- hover video previews
- gsap animations
- custom cursor
- responsive design
- modern ui

## built with

- html5
- css3
- javascript
- gsap
- scrolltrigger

## run locally

```bash
git clone https://github.com/helloayushhh/editkaro.in.git

cd editkaro.in
```

open `index.html` in your browser.
